# Todo List Web App

## Description
This is a simple Todo List web application built using **HTML, CSS, and JavaScript**.  
It allows users to manage daily tasks in an easy and interactive way.

The app was created with the help of **AI tools** to generate, refine, and debug the code.

---

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Completed)
- Tasks are saved in **localStorage**
- Responsive layout for mobile
- Clean and beginner-friendly code

---

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)
- localStorage API

---

## How to Use
1. Type a task in the input field
2. Press **Add** or hit **Enter**
3. Click on a task to mark it as completed
4. Click the **X** button to delete a task
